# shadcn MCP Notes

This starter uses the `npx shadcn@latest` CLI flow by default.

Only use shadcn MCP docs after a project has explicitly chosen to configure an
MCP server. Until then, use:

```bash
npx shadcn@latest info
npx shadcn@latest search
npx shadcn@latest docs button dialog select
npx shadcn@latest add button
```
