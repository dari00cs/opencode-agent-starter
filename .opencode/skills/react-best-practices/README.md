# React Best Practices

This is a vendored reference skill for React and Next.js performance,
reliability, and rendering guidance.

Use `SKILL.md` first. It routes the agent into the detailed rule files under
`rules/` when a specific concern applies.

This starter does not include the upstream build pipeline for regenerating the
compiled reference material.
