# Advanced And Optional Skills

These are useful, but not day-one essentials.

## Agent Orchestration

- `subagent-driven-development`: use when a written plan has independent tasks.
- `dispatching-parallel-agents`: use when there are multiple independent
  investigations or failures.

## Workspace And Repo Flow

- `using-git-worktrees`: create an isolated workspace when the user asks or the
  change is large enough to justify it.
- `finishing-a-development-branch`: run final verification and prepare handoff
  after implementation.

## Skill Maintenance

- `writing-skills`: create, edit, and test skills.

## Specialized Output

- `stitch-skill`: generate a Google Stitch-oriented `DESIGN.md` or semantic
  design-system spec. It is not for normal frontend coding.

## Extras Folder

The `extras/` folder contains workflows that are not active by default. Move an
extra into `.opencode/skills/` only after the project has the tools it needs and
the user wants that workflow.
