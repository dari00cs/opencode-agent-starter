# Skills

OpenCode discovers active skills from flat folders:
`.opencode/skills/<skill-name>/SKILL.md`.

Keep the active skill folders flat. Do not move them into category folders or
OpenCode may stop discovering them.

For humans, use these group guides:

- `_groups/start-here.md`: beginner route and daily defaults
- `_groups/superpowers.md`: planning, debugging, TDD, review, and verification
- `_groups/frontend-and-taste.md`: UI quality, taste, style flavors, React, and shadcn
- `_groups/advanced-and-optional.md`: subagents, worktrees, writing skills, Stitch, and extras

When using a skill, load the actual `SKILL.md`; these group files are only maps.
