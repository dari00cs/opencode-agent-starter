## Claude Code Notes

This repo is OpenCode-first. Claude Code should follow the same behavior
contract through `AGENTS.md`.

The grouped skills in `skills/` are readable source material here, not native
Claude Code project skills. If you want native Claude Code skill discovery too,
copy selected skills into `.claude/skills/<skill-name>/SKILL.md` later.

Use plan mode before risky edits. Prefer the included Superpowers and
frontend/taste skill instructions over repeating long prompts.
