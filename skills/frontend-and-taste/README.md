# Frontend And Taste

Skills for UI craft, visual taste, and frontend polish.

## Default UI Stack

- `frontend-skill`
- `taste-skill`
- `redesign-skill`

Use these together for most UI work.

## Optional Style Flavors

- `gpt-tasteskill`
- `minimalist-skill`
- `soft-skill`
- `brutalist-skill`

Use style flavors only when the user asks for that vibe. Otherwise default to
`taste-skill`.
